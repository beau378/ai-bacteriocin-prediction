# AI Bacteriocin Prediction

## Présentation du projet

Ce repository présente une version documentée de mon projet de stage de recherche portant sur l’utilisation de l’intelligence artificielle pour l’analyse de données biologiques liées aux bactériocines.

Le projet vise à explorer comment des modèles d’apprentissage automatique peuvent être utilisés pour analyser des données biologiques et contribuer à la prédiction de phénomènes liés à l’activité antibactérienne.

## Contexte

Les bactériocines sont des peptides ou protéines produits par certaines bactéries et capables d’inhiber la croissance d’autres bactéries. Leur étude présente un intérêt important dans un contexte de recherche d’alternatives aux antibiotiques.

Dans ce projet, l’objectif général était d’explorer des approches d’intelligence artificielle permettant d’analyser des données biologiques afin de mieux comprendre et prédire certains comportements associés aux bactériocines.

## Objectifs du projet

Les principaux objectifs du projet sont les suivants :

* structurer les données pour les rendre exploitables par des modèles d’intelligence artificielle ;
* analyser des données biologiques liées aux bactériocines ;
* développer un pipeline de traitement et de modélisation ;
* évaluer les performances des modèles prédictifs ;
* documenter la méthodologie suivie et les choix techniques réalisés.

## Méthodologie générale

La démarche méthodologique comprend plusieurs étapes :

1. compréhension du contexte biologique et des variables disponibles ;
2. préparation et nettoyage des données ;
3. transformation des données en formats exploitables pour l’apprentissage automatique ;
4. entraînement de modèles de machine learning ;
5. évaluation des performances des modèles ;
6. interprétation et synthèse des résultats.

## Technologies utilisées

Les principales technologies utilisées dans ce projet sont :

* Python ;
* pandas ;
* NumPy ;
* scikit-learn ;
* matplotlib ;
* Jupyter Notebook ;
* Git et GitHub.

## Confidentialité des données

Les données originales utilisées dans le cadre de ce projet ne sont pas incluses dans ce repository pour des raisons de confidentialité et de respect des règles liées au projet de recherche.
